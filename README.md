# Twitter Following Only - Firefox Extension

A Firefox extension that blocks the Twitter/X "For You" page and keeps you on the "Following" timeline. **Work in progress!**

# Problems
- Blocks the list of people any account is following
- Blocks some of settings?

### Installation (since I have not published it yet)

1. Open Firefox and navigate to `about:debugging`
2. Click "This Firefox" in the left sidebar
3. Click "Load Temporary Add-on..."
4. Navigate to this folder and select the `manifest.json` file
5. The extension will be loaded and active until you restart Firefox
